# 插件化设计方案

## 1. 什么是插件化设计方案？

插件化设计其实离我们非常近，日常使用到的构建工具像 webpack、vite 其实都是插件化模式

```ts
export default defineConfig({
    plugins: [
        vue(),
        vueJsx()
    ]
})
```

常见的就是在配置文件中通过 plugins 定义插件，本质上都是对既定的接口差异化实现，所以插件化我们可以理解是 **宿主容器在运行时有固定的对外接口，外部只需要重新实现相关接口就自定义容器运行时行为**。
