# H5 页面开发

## 移动端多媒体断点

- 小型设备（手机，< 768px）

```css
@media (max-width: 767.98px) {
}
```

- 中型设备（平板，≥ 768px 且 < 992px）

```css
@media (min-width: 768px) and (max-width: 991.98px) {
}
```

- 大型设备（桌面，≥ 992px 且 < 1200px）

```css
@media (min-width: 992px) and (max-width: 1199.98px) {
}
```

- 超大型设备（大屏桌面，≥ 1200px）

```css
@media (min-width: 1200px) {
}
```
