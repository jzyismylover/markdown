# echarts

可视化图表开源库

🔐 echarts 启动步骤

  1. 注册 DOM 作为容器
  2. 初始化 echarts 实例
  3. 配置项定义
  4. 将配置项注入 echarts 实例（ setOptions ）

因此关键是理解配置项！！！

# options

配置可以分为通用配置和特定图表配置

  • 通用配置

    • title：图表标题
    • tooltip：提示。主要指当悬浮或者点击后图表显示对应的数据
    • tooltox：工具按钮。包含导出、数据切换、放大缩小、数据视图、重置
    • legend：图例；主要用于筛选

  • 数据配置

    • xAxis：配置 x 轴
    • yAxis：配置 y 轴
    • series：配置图表数据、类型……

# scatter

散点图，用于呈现两组变量之前相关性。区别于折线图或者柱状图之类，x y 轴都是 value 轴，因此对应 series 要配置成数组

const options = {
  xAxis: {
    type: "value",
    scale: true,
  },
  yAxis: {
    type: "value",
    scale: true,
  },
  series: [
    {
      type: "scatter",
      data: [[]],
    },
  ],
};

# pie

饼图可以帮助快速了解不同分类的数据的占比情况

🔐 其中说明下圆环以及南丁格尔图的实现

  • 圆环：对应设置圆环的是 radius

    radius: ["50%", "75%"];
// 50% 对应的是内圆半径
// 75% 对应的是外园半径
    1
    2
    3

  • 南丁格尔图：直白说就是每个饼图区域半径大小都不一致（根据数据大小区分），对应的设置属性是 roseType，置为 true 即可

# gauge

仪表盘，呈现一个动作的得分情况

# display

对于显示 echarts 支持进行不同级别对上述属性进行配置

  • 主题：可从 echarts 官网自定义主题并下载成 js 的方式引入即可

    instance.init(dom, themeName);
    1

  • 调色板：调色板即为颜色配置

    • 主题级别
    • 全局级别
    • 局部级别

    可在不同级别对 color 进行配置，按照“就近原则”，配置越近优先级越高

  • 样式

    • itemStyle
    • lineStyle
    • areaStyle
    • label
